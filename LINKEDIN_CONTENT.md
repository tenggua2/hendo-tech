# LinkedIn 公司主页设置 — 全部内容（复制粘贴即可）

## 操作步骤

### 第1步：登录 LinkedIn
- 地址：https://www.linkedin.com/login
- 用您的账号登录

### 第2步：打开公司创建页面
登录后访问：https://www.linkedin.com/company/setup/new/

### 第3步：填写公司信息（复制粘贴以下内容）

**Company Name（公司名称）：**
```
Jiangsu Hengdu Phase Change Technology Co., Ltd.
```

**Website（网站）：**
```
https://tenggua2.github.io/hendo-tech/
```

**Industry（行业）：** 搜索并选择 `Building Materials`

**Company Size（公司规模）：** 选择 `51-200 employees`

**Tagline（标语）：**
```
World's First Screw-Extruded A2 Non-Combustible Aluminum Composite Panel
```

**Logo：** 用网站里的 `images/logo.png`（先下载到本地）
